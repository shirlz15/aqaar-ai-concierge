# Aqaar Frontend v1

Static, API-driven Aqaar AI Concierge frontend.

Data sources:

- `../Concierge-Backend-v1` APIs
- Backend data is loaded from `../KB-Acq` and `../Intelligence-Layer-v2`

The frontend does not modify backend, KB, or intelligence packages.

Run:

```powershell
npm run build
npm test
npm start
```

Default frontend URL:

```text
http://localhost:5173
```

Default backend URL:

```text
http://localhost:8080
```

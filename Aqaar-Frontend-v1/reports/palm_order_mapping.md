# Palm-Style Order Mapping for Aqaar

Palm deck was used only for module order and product-flow reference. No Palm content, colors, data, projects, pricing, or copy are used.

15. Runtime activity/events section

18. Download guide/brochure

20. Empty states with helpful suggestions

## Aqaar Data Policy

- Frontend reads data through `Concierge-Backend-v1` APIs.
- Backend reads from `KB-Acq` and `Intelligence-Layer-v2`.
- Missing data is displayed as `Contact Aqaar for details.`
- No hardcoded project, price, ROI, or ranking data is introduced.
